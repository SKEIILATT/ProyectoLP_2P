# Sistema de Análisis Educativo con IA (RAG)
### Materia: Lenguajes de Programación – ESPOL

**Integrantes:**
- Javier Gutiérrez
- Jair Palaguachi

**Descripción:**
Sistema inteligente capaz de responder preguntas complejas sobre rendimiento y abandono estudiantil utilizando RAG (Retrieval-Augmented Generation) con Ollama, LangChain y ChromaDB.

**Metodología:**
Desarrollo colaborativo completo. Ambos integrantes participan en el desarrollo Frontend, Backend y Ciencia de Datos.

**Módulos del Sistema:**
1. **Abandono Estudiantil:** Dataset UCI, API Laravel, Dashboard React.
2. **Rendimiento Académico:** Dataset Open University, API Laravel, Dashboard React.

**Tecnologías:**
- **Frontend:** React + TypeScript (Dashboards, Recharts)
- **Backend:** PHP/Laravel (API) + Python (RAG/Data Analysis)
- **IA:** LangChain, Ollama, ChromaDB
